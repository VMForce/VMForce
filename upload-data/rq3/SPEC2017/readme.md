Here is rq3 results on SPEC2017.
We use serial numbers to uniquely identify the conditional transfers in the program to facilitate the statistical results of the experiments.
Each folder contains the following files:
```
ct.txt - the conditional transfers in the program
rct.txt - the conditional transfers can be reached when running the program
orct_vmp.txt - the conditional transfers obfuscated by VMProtect
orct_themida.txt - the conditional transfers obfuscated by Themida
result_vmp.txt - the result of VMForce on the program obfuscated by VMProtect
result_themida.txt - the result of VMForce on the program obfuscated by themida
```
